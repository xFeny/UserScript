
# 依赖

需先安装「[视频自动网页全屏｜倍速播放]」才能使用。

# 说明

通用的页内视频悬浮小窗，当视频容器不在可视区域内，自动在右下角悬浮显示视频，视频在可视区域内时自动恢复。

### **自定义选项**：

**此站启/禁用悬浮小窗：** 默认为禁用，按需启用。

**设置小窗的宽高：** 可手动调整视频小窗的大小。

**小窗视口监测元素：** 优先使用配置的自定义元素，无配置则默认使用视频父容器。

- **解决的问题：** 
  - 与网站自动的小窗冲突时，通过自定义监测元素的方式绕过。
  - 默认监测元素的高度影响悬浮小窗的显隐。

**此站网址黑名单：** 网址前缀匹配，添加不需要显示小窗的页面

- 如：https://www.youtube.com/results,https://www.youtube.com/@
